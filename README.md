# 当作品について
当作品は、YouTubeのリンクを特定の場所に貼り付け、アップロードするサイトです。動画は自動で埋め込みされます。
また、アップロードした動画はローカルファイルに自動で保存されるので、毎度毎度動画をアップロードする手間を省いています。

# 使い方
## 動画をアップロードする
1. 観たい動画のリンクをコピーする or 手で入力する
2. 入力後、「追加」ボタンを押す
3. 「プレイリストに追加しました！」と緑背景の通知が来たら追加完了
## 動画を削除する
1. 動画選択の意義は自部分にある「✕」ボタンを押す
2. 「この動画をプレイリストから削除しますか？」でOKを押す
3. 「プレイリストから削除されました」と青背景の通知が来たら削除完了。
## プレイリスト作成
1. 「プレイリスト管理」までスクロールする
2. 「すべてのプレイリスト」を押す
3. 「新しいプレイリスト名」の部分に作りたいプレイリストの名前を入力
4. 「作成」or「新規作成」を押す
5. プレイリストに移動できたら作成完了
## プレイリスト削除
*現在、プレイリスト削除の機能は実装されていません。*

# 機能紹介
## プレイリスト機能
動画を指定のプレイリストにまとめる機能。自動再生はないが、現在開発中。
この機能では、アップロードした動画をまとめたりする際に使うのに役立ちます。いくらでも作成可能で、アップロード制限はありません。
#### 新機能開発中
* プレイリスト削除機能を開発中。完成次第順次更新予定。
* プレイリストをサイドバーから一覧表示できる機能を開発中。完成次第順次更新予定。
* 自動再生機能(ON/OFF付き)を開発中。完成次第順次更新予定。
## 保存機能
動画を「保存済み動画」に追加する機能。ボタン一つで保存が可能。
この機能では、アップロードした動画で特にお気に入りの動画を保存したりできます。保存数に制限はなく、いくらでも保存できます。
#### 新機能開発中
* 保存した動画をサイドバーから一覧表示できる機能を開発中。完成次第順次公開予定。
## 共有機能
YouTubeリンク・埋め込みコードのコピー、Facebook・X(旧Twitter)での共有が可能。現在その他でのサイトでの共有もできるように作成中。
この機能では、先述のサイトでの共有や、YouTubeリンクや埋め込みコードのコピーが可能です。共有数に制限はありませんが、XやFacebookなどで過剰にやりすぎるのはおかしい行為なので、止めましょう。
#### 新機能開発中
* 他サイトでの宣伝も可能にする機能を開発中。完成次第順次公開予定。
* サイドバーでも宣伝を可能にする機能を開発中。完成次第順次公開予定。

# 開発中の機能
## 履歴機能
サイドバーから確認可能の機能。アップロードした日時・動画ID・サムネイルが表示されるようにする予定。
完成後は、「履歴」というサイドメニューの部分から表示できるようにする予定。
#### <完成予定図>
[サムネ]
[動画ID]
[YYYY年MM月DD日]
(※月日が1桁の場合は下二桁目を0と置く予定です)
## アカウント機能
アカウントを作成・利用できる機能。現在の「Y」と書かれた部分に追加予定。
アカウント作成によって、機種が変わったり、Cookieを消したりしてもアップロード記録が残るなどの利点があるという理由での作成です。
完成後は、"ユーザー名(半角英数字のみ利用可能)"・"パスワード(8文字以上/記号制限なし)"でアカウント作成できるようにする予定です。
#### <完成予定図>
[アカウント作成・ログイン]
[ユーザー名記入欄]
[パスワード記入欄]
{作成/ログインボタン}
## 既存動画検索機能
既存動画(アップロード済み動画)を検索する機能。検索バーの部分で検索ができるようにする予定。
検索機能には開発期間が要すると思われるため、完成予定図は未定です。
